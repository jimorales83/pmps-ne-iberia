# Figures

The repository contains the redistributable main and supplementary figures used in the manuscript.

- Figures 3-6 are regenerated by `R/01_reproduce_rcarbon_figures.R` and `R/02_reproduce_oxcal_figures.R`.
- Figure 7 is regenerated by `R/04_reproduce_discussion_figure.R` from the documented archaeological and climatic source tables.
- Figure 1 is a QGIS composition using EMODnet Bathymetry DTM 2022, Natural Earth 1:10m coastline/rivers and ICGC elevation/shaded-relief data. EMODnet and ICGC products are used under CC BY 4.0; Natural Earth data are in the public domain.
- Figure 2 is an author-generated conceptual workflow diagram.
- Supplementary Figures S1-S5 are generated by the radiocarbon workflow. The default guided run produces S1; set `RUN_ALL_RCARBON_CHECKS <- TRUE` in `R/run_reproduction.R` to regenerate S2-S5.

Figure 8 is not redistributed because panels adapted from Maroto et al. (1996) and Mora et al. (2018) require a separate rights check. This omission does not affect the analytical reproduction workflow.

- EMODnet terms: https://emodnet.ec.europa.eu/en/terms-use-emodnet-online-services-data-and-data-products
- Natural Earth terms: https://www.naturalearthdata.com/about/terms-of-use/
